1. Download & Self-Host

Use tools like yt-dlp to download the video and host it yourself:

# Install yt-dlp
brew install yt-dlp

# Download Instagram video
yt-dlp "https://www.instagram.com/reel/C8Cp5VvvNG6/"

# Then use HTML5 video
<video controls width="350" height="622">
  <source src="your-video.mp4" type="video/mp4">
</video>

2. Use Video CDN Services

  • Cloudinary - Free tier, automatic optimization
  • Mux - Professional video hosting
  • Bunny CDN - Affordable video streaming
  • AWS S3 + CloudFront - Scalable but more complex

3. YouTube Instead

Upload to YouTube and use their cleaner embed:

<iframe 
  width="350" 
  height="622" 
  src="https://www.youtube.com/embed/VIDEO_ID" 
  frameborder="0" 
  allowfullscreen>
</iframe>

4. Next.js Video Component

For your React app, create a proper video component:

const VideoPlayer = ({ src, poster }) => (
  <video
    controls
    playsInline
    className="w-full h-auto rounded-lg shadow-lg"
    poster={poster}
  >
    <source src={src} type="video/mp4" />
  </video>
);
